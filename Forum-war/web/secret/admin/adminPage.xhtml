Administrator Features


  1. Create Administrator: Creates new users who have immediate administrative privileges.

  2. View All Users: View a list of all users (active, blocked, unblocked).

    1. 'View' Icon: Administrators can view the profile of each individual user.

    2. 'Block' Icon: Administrators can block users from using the forum.

    3. 'Unblock' Icon: Administrators can unblocked previously blocked users to become normal users.

    4. 'Promote' Icon: Administrators can promote normal users to administrators.

    5. 'Demote' Icon: Administrators can demote administrators to normal users.

  3. Create Forum: Create a Forum with a Forum Category and Description.

  4. View All Forums: View all the forums that are available on the Main Forum.

    1. 'View Forum' Icon: Administrators can view the selected forum.

      1. 'View Thread' Icon: Administrators can view the selected thread.

        Administrators can further edit and delete any posts using the 'Edit' and 'Delete' icons after clicking on 'View Thread'.

      2. 'Edit Thread' Icon: Administrators can edit the thread title and description of any thread.

      3. 'Delete Thread' Icon: Administrators can delete any thread.

      4. 'Close Thread' Icon: Administrators can close any thread that is currently active.

      5. 'Open Thread' Icon: Administrators can open any thread that is currently closed.

    2. 'Edit' Icon: Administrators can edit the forum category and description.

    3. 'Delete' Icon: Administrators can delete the forum.
